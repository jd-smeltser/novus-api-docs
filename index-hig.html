How It Works

The Novus API uses a simple two-step authentication process designed for Make.com integration.

Key Insight: Tokens expire in 5 minutes to ensure security. Our documentation shows you exactly how to handle this gracefully in your workflows.

Authentication Flow

Simple Two-Step Process 
1. Request Token
   POST /Token (with credentials in headers)
   ‚Üí Returns: Authentication token

Key Features

  • Secure Authentication: Token-based access with automatic expiration
  • Complete Member Data: Demographics, policies, and contact information
  • IP Security: Whitelisted access for enhanced protection
  • Environment Support: Separate QA and Production endpoints
